# Nurfachmi

Nurfachmi adalah sebuah komunitas atau wadah yang memiliki visi dan misi untuk menyediakan aplikasi siap pakai dan gratis untuk kepentingan maslahat.

Berikut beberapa projek yang sedang dikembangkan

## Simasjid

Sistem informasi rumah ibadah (repositori menyusul)

## Siponpes

Sistem informasi pondok pesantren modern (repositori menyusul)

## Siperpus

Sistem informasi perpustakaan (repositori menyusul)
